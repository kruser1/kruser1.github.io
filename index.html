Identifying Regional COVID-19 Presence Early with Time Series Analysis

Type: Research Publication
Skills: R, Time Series Analysis
Date: Summer 2020
Paper
Video Presentation

In this project, I used data from the CDC's Influenza-like Illness Surveillance Network to find evidence of COVID-19 presence in the US in December 2019. Several months after publication, my findings were corroborated by CDC scientists in a report that made national headlines.

Visualizing Catchment Analysis with SafeGraph Patterns

Type: Tutorial
Skills: Python, Visualization
Date: Summer 2020
Notebook

The goal of this project was to clearly demonstrate a value area of SafeGraph's data to those unfamiliar with the data. I created a dynamic map visualization to show where a business's visitors are coming from. The resulting Google Colab notebook was featured in one of SafeGraph's company blog posts.

A Framework for Association Rule Learning with Social Media Networks

Type: Class Project
Skills: Python, Association Rule Learning, Data Mining
Date: Fall 2020
Paper
Video Presentation

While association rule learning is commonly known for market basket analysis, this project demonstrates an innovative application that treats Twitter users as trips to the store and followers as grocery items. The result allows for analysis and comparison of follower/following networks, including a unique look at social media "echo chambers."

Grobner Bases of Neural Ideals

This 2018 publication resulted from a 2015 research internship in Hilo, Hawaii. We applied algebraic geometry to better understand the structure of neural codes.




Academic Administrator Dashboard

As part of a class project for my Data Analytics course, I created a Power BI dashboard for an academic administrator. The dashboard allows the user to quickly get information on each department.




Krobel Corp Startup

From January 2018 to March 2020, I cofounded and ran a small startup that used thermal imaging and machine learning to boost piglet health. Ultimately, the company was acquired by one of our investors in a minor acquisition.




Political Census (Forthcoming)

This Google Colab notebook demonstrates the creation of a political census. Joining datasets on voting results, polling locations, and human movement, I estimate political affiliation at the Census Block Group-level—a huge improvement over the current county-level estimates.




YoY Brand Performance Estimation

This project used SafeGraph's human movement data to estimate nationwide year-over-year brand performance. As the underlying panel changes over time, I compared various normalization techniques to best fit the "ground truth YoY" from the brand's earnings reports.




Google Civic API Crawler

Polling location data is not widely available at the state or national level. The Google Civic API is the next best thing; it returns the polling place(s) corresponding to a given address. For this time-crunched project, I set up an API crawler to build a dataset of US polling places (~60% coverage) using sample addresses.
